Remote Work Setup

Whether your staff work from home, the road or multiple sites, I'll set up the tools and access they need to work securely and without headaches.

What's included

  • VPN setup for secure remote access to your office network
  • Remote desktop configuration (Windows RDP, AnyDesk, TeamViewer)
  • Microsoft 365 setup — email, Teams, SharePoint and OneDrive
  • Cloud file access so staff can reach shared files from anywhere
  • Multi-factor authentication to keep remote access secure
  • Laptop and device setup for home or mobile workers
  • Troubleshooting connection and access issues

Who this is for

  • Businesses with staff working from home full or part time
  • Teams spread across multiple locations
  • Business owners who need to access their systems while away
Remote access done poorly is a security risk. I'll make sure access is locked down properly — not just convenient, but safe.
